Public pokemon data

Static JSON API

Base URL: https://dannyverdel.github.io/pokemon-data
  • /data/data.json
      Get all the sets from the pokemon trading card game. Each set contains an 'cards' property containing all the cards related to that set.
  • /data/rarities.json
      Get all the unique rarities.
  • /data/subtypes.json
      Get all the subtypes.
  • /data/supertypes.json
      Get all the supertypes.
  • /data/types.json
      Get all the different pokemon types.
  • /data/sealed-products.json
      Get sealed products and their prices. source
  • /data/pull-rates.json
      Get detailed pull rates and card order information for each pack. source
Updates at 00:00 utc every day.
